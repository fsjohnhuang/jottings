• 透明(Opacity)
    1. CSS3
      • opacity:属性值，属性值，属性值范围为0-1。1为无效果，0为透明。
      • rgba(r,g,b,a)，若是纯色的透明可使用rgba(r,g,b,a)设置透明状态。通过该方式设置的透明状态仅作用与指定的样式本身（如border,background-color,color等），而不会作用与元素整体及其子元素上。
        • r，红色，取值范围0-255
        • g，绿色，取值范围0-255
        • b，蓝色，取值范围0-255
        • a，透明度，取值范围0-1
    2. IE5~8
      • filter:alpha(opacity=属性值)，属性值，属性值范围为0-100。100为无效果，0为透明。
    Opacity
    RGBA
  • 粽褐色(Sepia)
    1. CSS3中通过filter:sepia(属性值)，属性值范围为0-1或0-100%。0为无效果，1或100%为最大效果。
    Sepia
  • 灰度图(Grayscale)
    1. CSS3中通过filter:grayscale(属性值)，属性值范围为0-1或0-100%。0为无效果，1或100%为最大效果。
    Grayscale
  • 高斯模糊(Blur)
    1. CSS3——通过filter:blur(属性值)，属性值为pixel值，表示以元素中心为圆心，模糊的半径范围。0为无效果。
    2. IE滤镜——filter:progid:DXImageTransform.Microsoft.Blur(PixelRadius=不带单位的半径值, MakeShadow=false)
    3. SVG——filter:url(blur.svg#blur)
    Blur
  • 底片效果(Invert)
    1. CSS3——通过filter:invert(属性值)，属性值范围为0-1或0-100%。0为无效果，1或100%为最大效果。
    Invert
  • 饱和度(Saturate)
    1. CSS3——通过filter:saturate(属性值)，属性值为百分比或数字,值大于等于0即可。1为无效果，0为灰度图。
    Saturate
  • 对比度(Contrast)
    1. CSS3——通过filter:contrast(属性值)，属性值为百分比或数字,值大于等于0即可。1为无效果。
    Contrast
  • 亮度(Brightness)
    1. CSS3——通过filter:brightness(属性值)，属性值为百分比或数字,值大于等于0即可。1为无效果。
    Brightness
  • 色相旋转(Hue Rotate)
    1. CSS3——通过filter:hue-rotate(属性值)，属性值为0deg-360deg,默认值为0deg。
    Hue Rotate
  • 阴影(Drop Shadow)
    1. CSS3
      • drop-shadow滤镜(仅Chrome支持)，通过filter:drop-shadow(x-offset y-offset 阴影模糊半径 阴影颜色)
        • 注意点：
          1. 投影的外观受到border-radius样式的影响。
          2. :after和:before等伪元素继承投影的效果
        • REFERENCE
      • box-shadow样式，通过box-shadow: 投影方式(inset) X轴偏移量(x-offset) Y轴偏移量(y-offset) 阴影模糊半径(blur-radius) 阴影扩展半径(spread-radius) 阴影颜色(color)，可同时使用各个投影，多个投影设置间使用逗号(,)分隔。
        • 投影方式(inset)，可选属性（默认为外阴影），设置为inset时表示内阴影。
        • X轴偏移量(x-offset)，正值表示阴影位于对象的右边，负值表示阴影位于对象的左边。
        • Y轴偏移量(y-offset)，正值表示阴影位于对象的底部，负值表示阴影位于对象的顶部。
        • 阴影模糊半径(blur-radius)，可选属性（默认为0，没有模糊效果），值范围>=0。
        • 阴影扩展半径(spread-radius)，可选属性，正值表示阴影扩展，负值表示阴影缩小。
        • 颜色(color)，可选属性(默认值又浏览器决定, Webkit内核的为透明)
        • 注意点：
          1. 投影的外观受到border-radius样式的影响。
          2. :after和:before等伪元素不继承投影的效果
        • REFERENCE CSS3 box-shadow Box-shadow常被遗忘的一个参数——“spread” CSS3 制作Drop Shadow效果
      • IE6~8
        • IE滤镜， filter:progid:DXImageTransform.Microsoft.Shadow(color=颜色值,direction=阴影角度,strength=阴影半径)
          • 颜色值必须使用#000000格式，而不能使用#000的格式
          • 阴影角度，取值范围[0,45,90,135,180,225,270,315,360]
          • 阴影半径，取值范围>=0
          • 注意点：
            1. 必须配合backgroud才有效果

  • 圆角(border-radius)
    1. CSS3, 通过border-radius: top-left-radius top-right-raduis bottom-right-radius bottom-left-radius [/ top-left-radius top-right-raduis bottom-right-radius bottom-left-radius]
      • /前设置各个角的水平外半径,/后设置各个角的垂直外半径。若仅有/前则一次性设置各个角的水平、垂直外半径。
      • 当属性值小于4时，则才采用对角设置法。如设置为border-radius:5px 10px则表示top-left和bottom-right的外半径为5px，而top-right和bottom-left的外半径为10px
      • 逐个角设置
        border-top-left-radius: 水平半径 垂直半径;
        border-top-right-radius: 水平半径 垂直半径;
        border-bottom-left-radius: 水平半径 垂直半径;
        border-bottom-right-radius: 水平半径 垂直半径;
      • 内半径， 内半径=外半径-边宽 >= 0 ? 外半径-边宽 : 0。因此当且仅当外半径大于边宽时，才能看到内半径。
      • 注意：
        1. 对img元素使用border-radius设置含内半径的圆角，Safari5.0、Google Chrome 10.0、Opera11.1和FF4.0以下会无法出现内半径，而FF4.0则会出现内半径
        2. 对table元素使用border-radius设置圆角，当table的border-collapse为collapse时，无法效果。仅border-collapse为separate时才有效果
      • REFERENCE CSS3的圆角Border-radius

  • 线性渐变(Linear Gradient)
    1. CSS3, 通过属性值linear-gradient(angle, color stop, color stop[, color stop]*)设置线性渐变
      • angle,用于设置起始位置，可设置数字或预设值。数字取值范围：0-359deg，预设值取值范围：[left,right,top,bottom, center, top left, top right, bottom left, bottom right, left center, right center]。IE10下的取值范围为:[left, top]。Chrome下没有left center, right center和center。默认值为top。
      • color,渐变颜色
      • stop,渐变点，用于设置color的起始位置，值为百分数，小于0%或大于100%时表示颜色的起始位置在可视区域之外。
      • REFERENCE 《HTML.5与CSS.3权威指南》没有该内容 《HTML5秘籍》8.4.5.渐变盒子 《响应式Web设计：HTML5和CSS3实战》6.3.背景渐变，6.4.背景图渐变 介绍repeating-linear-gradient和repeating-radius-gradient CSS3 的线性渐变
      • 工具
    2. IE5.5~9，IE5.5~7通过滤镜filter:progid:DXImageTransform.Microsoft.gradient(startcolorstr='起始颜色',endcolorstr='结束颜色',GradientType=1)，IE8~9通过滤镜-ms-fitler:"progid:DXImageTransform.Microsoft.gradient(startcolorstr='起始颜色',endcolorstr='结束颜色',GradientType=1)" 
                注意：元素必须hasLayout后滤镜才会生效.(启动hasLayout的方式：设置height、width属性，position为absolute，writingMode为tb-rl，或contentEditable为true).
          GradientType,0(默认值，从top到bottom渐变),1(从left到right渐变)
          endcolorstr和startcolorstr值格式为#AARRGGBB,其中AA为透明度的十六进制表示形式。endcolorstr默认值为#FF000000,startcolorstr默认值为#FF0000FF.
          endcolor和startcolor为十进制的整数类型,取值范围0~4294967295

  • 径向渐变/放射性渐变(Radial Gradient)
    1. CSS3,通过属性值radial-gradient(圆心位置, 渐变形状 渐变大小,color stop, color stop[, color stop]*)来设置

      • 渐变大小, 使用预设值closest-side 或 contain(以距离圆心最近的边的距离作为渐变半径);closest-corner(以距离圆心最近的角的距离作为渐变半径);farthest-side(以距离圆心最远的边的距离作为渐变半径);farthest-corner 或 cover(以距离圆心最远的角的距离作为渐变半径);
